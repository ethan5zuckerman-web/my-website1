# Ethan Zuckerman — Portfolio Website

Personal portfolio site built with HTML, CSS, and JavaScript. Hosted on GitHub Pages.

**Live site:** https://ethan5zuckerman-web.github.io/portfolio/

## About

Single-page portfolio showcasing real estate experience, education, and professional background. Built to be fast, responsive, and employer-ready.

## Structure

```
index.html   — Page content and structure
style.css    — Styling, layout, responsive breakpoints
script.js    — Scroll animations, mobile nav, active state
headshot.jpg — Professional photo
favicon.svg  — Browser tab icon
```

## Features

- Responsive design (mobile, tablet, desktop)
- Smooth scroll animations via IntersectionObserver
- Two-column hero with headshot
- Timeline layout for work experience
- Proper meta description and Open Graph tags

## Built With

- HTML5 / CSS3 / Vanilla JavaScript
- Google Fonts (Playfair Display, Inter)
- GitHub Pages for hosting
